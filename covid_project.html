• Project Summary

For this project, I gathered USA COVID-19 time series data from Johns Hopkins University, including daily and total cases and deaths, as well as their locations. The code is written entirely in Python, and relies on pandas and numpy to manipulate the data, cartopy to map the cases/deaths, and matplotlib to create and animate the GIF.

Prior to this project, I had never done any work with COVID-19 data, but given the pressing nature of the pandemic, I thought it was a good dataset to branch out into. If you would like to see the full code for the project, here is a link to my GitHub repository: https://github.com/jakesanghavi/COVID-19_Map
